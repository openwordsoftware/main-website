ALPHA RELEASE TESTING

Thank you for agreeing to take time to assist with this project. The “MessageHub for Desktop” project’s goal is to build a desktop application (Windows based at first) utilizing the Messagehub’s database to make advanced searching of the Message easy for as many language groups as possible. It aims to at first be on par with Folio Views in terms of features, but eventually surpass it. Your feedback and input into this project is extremely valuable. Feature and design suggestions along with bug detection and testing is large contributing factor to the success of a software development project.

The 1.0 release will be English and Spanish only (although it will be capable of using any language infobase from the Message Hub)

This pre-release is to gain feedback on project direction as well as specific feedback on features and bugs.

At this point many major features have been implemented. Including but not limited to:

  • Browsing and viewing Messages and Bible
  • Copy and paste out of Message and Bible
  • Searching Message and Bible
  • Search term highlighting and jumping into text
  • Live hit totals when searching
  • Search results paging
  • Cross references from Bro Mamalis
  • Changing basic settings on documents
  • Settings persist across sessions

I would like to get your input on:

  1. The overall direction of the project.
  2. The overall design, window docking, treeview menu etc.
  3. What your ideal search experience would be like
    • Do you prefer live hit counts for each word?
    • Do you prefer word prediction?
    • How would you like search scoping to work? (ie limiting search to just one Message or a single year)
  4. Show stopping bugs
  5. General user experience feedback

Minimum System Requirements

Windows 10 (version 1607 "Anniversary Update" or newer)
or Windows 8.1
or Windows 7 SP1 with Platform Update
1GHz processor (dual-core recommended)
1GB of RAM

Message Hub for Desktop depends on Microsoft’s .NET Framework 4.5.1 or greater, which is automatically installed if it isn’t already on the system.

Message Hub for Desktop uses your hard drive to store infobase files that are included in the installer and any that you might download. Because of this, actual disk space requirements will depend on the number of infobases you install. However, the largest infobases are still less than 100MB each.
