Message Hub for Desktop

Powerful application to search the Holy Bible and the sermons of William Marrion Branham.

 

Supports Windows 7, 8.x, 10

Languages

Message Hub for Desktop supports downloading and searching all 80 languages supported by the Message Hub, including the Bible translated into each language.

For a complete list of all messages and languages visit http://messagehub.info

The installer and application menus currently support English and Spanish

Search

Message Hub for Desktop supports advanced text searching of both the Bible and Message including; phrase, wildcard, any word, and all word searches.

Cross References

Message Hub for Desktop supports cross references between Bible verses and sermon paragraphs.

The built in cross reference database is supplied by Rev David Mamalis.
