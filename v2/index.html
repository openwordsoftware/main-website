Cross Platform

In addition to Windows 10/11 support, OMV 2.0 now supports Mac OSX.

Languages

OMV 2.0 supports downloading and searching all 80 languages supported by the Message Hub, including the Bible translated into each language. For a complete list of all messages and languages visit http://messagehub.info

Updated User Interface

The user interface has been completely redesigned including an intuitive search builder.

Cross References

OMV 2.0 supports cross references between Bible verses and sermon paragraphs. The built in cross reference database is supplied by Rev David Mamalis.

User Notes

OMV 2.0 allows users to create notes attached to sermon paragraphs and/or Bible verses.

Subject Tags

Create tags for different subject, then attach those tags to sermon paragraphs and/or Bible verses.
