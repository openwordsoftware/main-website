Open Message View 2.0 Beta (Windows 10/11 & Mac OS)

System Requirements

  • Windows 11, Windows 10 (version 1607 “Anniversary Update” or newer), or Mac OSX v13+
  • OpenMessageView uses your hard drive to store infobase files that are included in the installer and any that you might download. Because of this, actual disk space requirements will depend on the number of infobases you install. However, the largest infobases are still less than 100MB each

Open Message View 1.0 (Windows 7/8)

System Requirements

  • Windows 11, Windows 10 (version 1607 “Anniversary Update” or newer), or Windows 8.1, or Windows 7 SP1 with Platform Update
  • OpenMessageView depends on Microsoft’s .NET Framework 4.7.2 or greater, which is automatically installed if it isn’t already on the system.

Minimum Machine Requirements

  • 1GHz processor (dual-core recommended)
  • 1GB of RAM
  • OpenMessageView uses your hard drive to store infobase files that are included in the installer and any that you might download. Because of this, actual disk space requirements will depend on the number of infobases you install. However, the largest infobases are still less than 100MB each
